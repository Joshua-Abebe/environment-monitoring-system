MySQL OWNS MEASURMENTS AND WILL BE USED FOR TIME SERIES ANALYSIS


NORMALIZED RELATIONAL DATABASE:

- Three tables:
           - SENSORS
           - ROOMS
           - READINGS

TABLE 1: SENSORS

- sensor_id
- sensor_type
- unit
- location_id (foreign key from ROOMS)

TABLE 2: ROOMS

- location_id
- location_name
- location_type

TABLE 3: READINGS (this is where the sensor data comes from)

- reading_id
- sensor_id (foreign key from sensors
- value
- timestamp
