DB-B1: Environmental Monitoring** as my Database Module B project topic.

Reason: I chose this project because it allows me to explore MQTT communication and compare relational, NoSQL, and graph databases in a simple real-time monitoring system using Python.

The planned system will include:

* MQTT protocol for real-time data publishing and subscription between simulated sensors and the application.
* Python for implementing the sensor data simulator and data processing pipeline.
* MySQL for storing structured sensor readings in a relational database.
* MongoDB for storing raw sensor data in JSON/document format.
* Neo4j for modeling relationships between sensors, locations, and measurements using a graph database.
